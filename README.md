# rock-paper-scissor-game
ROCK-PAPER-SCISSORS GAME (Python)

This is a simple command-line Rock-Paper-Scissors game written in Python.
The user plays against the computer, which chooses randomly.

--------------------------------------------------
FEATURES
--------------------------------------------------
- User can choose Rock, Paper, or Scissors.
- The computer randomly selects its choice.
- ASCII art is displayed for both user and computer choices.
- The game announces the winner: user wins, computer wins, or draw.

--------------------------------------------------
HOW THE GAME WORKS
--------------------------------------------------
1. The user is prompted to enter a number:
   - 0 for Rock
   - 1 for Paper
   - 2 for Scissors
2. The program validates the user input.
3. The computer randomly chooses a number between 0 and 2.
4. Both choices are displayed using ASCII art.
5. The game compares choices and prints the result:
   - User wins
   - Computer wins
   - Draw

--------------------------------------------------
REQUIREMENTS
--------------------------------------------------
- Python 3.x
- No external libraries needed

--------------------------------------------------
HOW TO RUN
--------------------------------------------------
1. Open terminal or command prompt.
2. Navigate to the project folder.
3. Run the program using:

   python myfile.py

4. Follow the on-screen instructions to play.
